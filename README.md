# dtMoney Ignite

Aplicação desenvolvida no Capitulo 2 do Ignite da Rocketseat, onde foi abordado conceitos de ReactJs como:
  * Estado, 
  * Imutabilidade, 
  * Componentização, 
  * Propriedades, 
  * Type Script,
  * Hooks,
  * Context
  
# Start

Install all dependencies: $ yarn;

Initialize the fake api: $ yarn server;

Up frontend: $ yarn start
